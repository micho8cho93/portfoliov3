# Physical Layer

The physical layer is the actual path where raw bit signals travel through material media.

<InternetLayerVisual layer="physical" analogyLabel="Roads, tunnels, and air lanes carrying the same cargo bits." />

## Real-World Analogy

Imagine one cargo type moving through three channels:

- Road lanes
- Tunnel lanes
- Air lanes

The package is the same, but the travel method changes by channel.

## Components In This Layer (From the PDF Definitions)

- `Fiber`: Bit signals carried as light pulses.
- `Copper`: Bit signals carried as electrical changes.
- `Wireless`: Bit signals carried as radio wave patterns.

## What The Loop Visual Shows

Three synchronized lanes animate signal pulses, showing how one bit stream can be represented as light, electrical pulses, or radio transmissions.
