# Network Layer

The network layer moves packets between networks, deciding where each packet goes next on its trip.

<InternetLayerVisual layer="network" analogyLabel="Postal routing across cities by destination address." />

## Real-World Analogy

Think of a postal system across many cities. Workers read destination addresses and pass each package to the next center until it reaches the target city.

## Components In This Layer (From the PDF Definitions)

- `IP`: Provides addressing and forwarding rules so routers can send packets hop by hop across interconnected networks.

## What The Loop Visual Shows

The node graph represents routers. A highlighted packet marker moves from node to node while the graph rotates, showing continuous forwarding and possible reroutes.
