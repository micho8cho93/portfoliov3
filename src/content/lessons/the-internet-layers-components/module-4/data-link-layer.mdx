# Data Link Layer

The data link layer handles local delivery on one network segment using frames and switching behavior.

<InternetLayerVisual layer="data-link" analogyLabel="Apartment mailroom sorting framed packages to local units." />

## Real-World Analogy

Imagine an apartment mailroom. Each package has a local unit label, and the mailroom clerk routes it to the right mailbox in the same building.

## Components In This Layer (From the PDF Definitions)

- `Frames`: Structured local-delivery units that carry data and local addressing details for one network link.

## What The Loop Visual Shows

Frame cards travel through a switch-like hub and then branch to specific local devices, repeating the local sorting cycle.
