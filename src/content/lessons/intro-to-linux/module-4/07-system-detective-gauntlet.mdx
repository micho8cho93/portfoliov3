# Mission Brief

You are the on-call investigator for a Linux server with suspicious behavior. Logs and clues are spread across multiple directories, and only a complete evidence trail unlocks the final code.

## Challenge Type

This is an **advanced forensic challenge**. You must prioritize command order, extract only relevant lines, and compare conflicting files before submitting your conclusion.

## Success Criteria

- You produce a concise incident summary from command output.
- You identify the correct final code/flag.
- You explain how at least one system command and one search command contributed to the answer.
